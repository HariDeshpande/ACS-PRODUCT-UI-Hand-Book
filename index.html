Few basic good practices:

Never use/write inline style. This will make our layout nonresponsive. Always align code when writing anything. Do try to (copy) & use HTML codes/CSS classes which are already available & in use, on other pages, if a similar UI is needed. This will help us to avoid creating CSS classes with the same properties but different names.

First thing is that we are using Bootstrap V4.6; so in V4.6 the row-column grid system has negative margins hence we should follow some basics.
Always use row > col, never use col without wrapping it in a row or don't use a row which does not have div with “col-” class inside it.


CORRECT WAY, row-col code should be written like this: 

    <div class="row">
        <div class="col-md-**"></div>
    </div>

Or if row inside a row, it should always be (row should not be enclosed directly by another row) 

    <div class="row">
        <div class="col-md-**">
        </div>
        <div class="col-md-**">
            <div class="row">
                <div class="col-md-**"></div>
            </div>
        </div>
    </div>

INCORRECT WAY, row inside col which is not wrapped in a row & no col used in inner row 

    <div>
        <div class="col-md-12">
            <div class="row">
                date/text/form
            </div>
        </div>
    </div>

    <div>
        <div class="row">
            date/text/form
        </div>
    </div>
    <div class="row col-12">
        date/text/form
    </div>

Avoid div inside div without any class or without required 

    <div>
        <div>
            <div class="row">
                <div class="col-md-**">
                    Some form/ Content
                </div>
            </div>
        </div>
    </div>

Below is way to add expansion panel:

(Note: default expansion panel is with primary background color [which is blue in our product] & style defined; so, to override & use it as per common expansion panel UI guide, with white background which is most used style; enclose the panel code inside a div with same class name (listview__reset-expansion-panel) mentioned below)

Use the below Example template Ex:

    <div class="listview__reset-expansion-panel">
        <!--(Note: With the help of [collapsed] flag = true, which is by default collapsed i.e. [collapsed]="true"; if this is set as false then by default the panel will be opened; BELOW LINE)-->
        <igx-expansion-panel [collapsed]="true">
            <!--(Note: Whith the help of this property [iconPosition]="'right'", the arrow for open/close panel can be positioned on left end or right end; BELOW LINE)-->
            <igx-expansion-panel-header [iconPosition]="'right'">
                <igx-expansion-panel-title>
                    The title or the data which will be shown by default & on click of icon at the end/start the panel will expand will go here. It can be row + col or para, text, heading tags anything.
                </igx-expansion-panel-title>
            </igx-expansion-panel-header>
            <igx-expansion-panel-body>
                This is the by default hidden data holder it will be shown if the pannel is expanded by clicking on arrow on the END/STSRT of the above header. It can be row+col or grid, table, form or para, text, heading tags any thing.
            </igx-expansion-panel-body>
        </igx-expansion-panel>
    </div>
